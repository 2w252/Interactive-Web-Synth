# Interactive Web Synth

Это легкий и функциональный синтезатор, работающий прямо в вашем браузере. Проект разработан с использованием **Web Audio API** и демонстрирует возможности генерации звуковых волн в реальном времени.

## 🌟 Возможности
*   **Осциллятор:** Поддерживает 4 типа звуковых волн (Sine, Square, Sawtooth, Triangle).
*   **Огибающая (ADSR):** Регулировка времени атаки (Attack) и затухания (Release) для изменения характера звука.
*   **Интерактивный визуализатор:** Осциллограф на базе HTML5 Canvas отображает звуковую волну в реальном времени.
*   **Управление:** Играйте кликами мыши по клавишам пианино или с физической клавиатуры (клавиши: `A W S E D F T G Y H U J K`).

## 🛠️ Запуск проекта
Проект написан на чистом HTML/CSS/JS (Vanilla JS) и не требует сборки. 

Просто откройте файл `index.html` в любом современном браузере!
