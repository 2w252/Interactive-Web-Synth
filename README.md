# Interactive Web Synth

Это легкий и функциональный синтезатор, работающий прямо в вашем браузере. Проект разработан с использованием **Web Audio API** и демонстрирует возможности генерации звуковых волн в реальном времени.

Код разделен на три независимые технологии для удобства чтения и масштабирования:
1.  **`index.html`** — разметка элементов управления, визуализатора и клавиатуры.
2.  **`style.css`** — стилизация интерфейса (Neon/Dark, Glassmorphism) и полная адаптивность под любые устройства.
3.  **`app.js`** — логика звукового движка (генерация нот, ADSR-огибающая, LFO-модуляция, Delay-эффекты и визуализатор).

## 🌟 Возможности
*   **Осциллятор:** Поддерживает 4 типа звуковых волн (Sine, Square, Sawtooth, Triangle).
*   **Мощные эффекты:** Низкочастотный фильтр (Lowpass) с частотой среза и резонансом, Delay (Эхо) и LFO (Вибрато).
*   **Пресеты:** Space Lead, Fat Bass, Retro Game.
*   **Адаптивность:** Полная поддержка мобильных телефонов, планшетов и ПК во весь экран.
*   **Управление:** Мышь/тач-панель или клавиши клавиатуры (`A W S E D F T G Y H U J K`).

## 🛠️ Запуск проекта
Просто откройте файл `index.html` в любом современном браузере!
